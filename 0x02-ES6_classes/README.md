# ALX Frontend JavaScript Project

## Project Description
This project implements various classes using ES6 syntax and demonstrates inheritance, getters, setters, and other object-oriented programming principles.

## Setup
1. Install NodeJS 12.11.x
2. Clone the repository
3. Run `npm install` to install dependencies

## Running Tests
- Use `npm run test` to run tests.
- Use `npm run full-test` to run tests with linting.

## Classes Implemented
- ClassRoom
- HolbertonCourse
- Currency
- Pricing
- Building
- SkyHighBuilding
- Airport
- HolbertonClass
- StudentHolberton
